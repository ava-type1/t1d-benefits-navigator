## üéØ How It Works

1. **User Profile** - Enter income, household size, state, insurance status
2. **Supply Selection** - Choose needed diabetes supplies (insulin, CGM, pump, etc.)
3. **Program Matching** - Algorithm scores eligibility (1-10) for all programs
4. **Detailed Guidance** - Step-by-step application process for each program
5. **Direct Application** - Links to official manufacturer and nonprofit applications
